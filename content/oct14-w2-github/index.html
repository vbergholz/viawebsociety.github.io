Basic commands (all you need to know)

git...

  • status - shows what you've added/changed/deleted/staged since last commit
  • add - stages any changes you've made (so they can be committed)
  • commit - backs up all staged changes
  • push - sends commits to the remote repository (makes the remote repo up-to-date with yours)
  • pull - pulls commits from the remote repository (makes your repo up-to-date with the remote)
  • clone - copies a whole remote repository (like off of GitHub)

Simplest Git workflow

  • git status
  • git add -A
  • git commit -m "Write a commit message here"
  • ...rinse and repeat...
  • git push

Intro to GitHub

What is it?

  • Created in 2008 to simplify code sharing
  • Cloud-based code repository
  • Works with git
  • Used for control-version, hosting, working in team...

How is it working?

  • Local git, 3 states : Modified, indexed (add), commited (commit)
  • When you push your commits : Your new code is saved on your github repo

How to start? (the easy way)

  • First of all, set up an account !
  • Create a repository
  • Add a README.md and a .gitignore (optional)
  • Clone your online repository on your local git (git clone url_of_your_repo [a_folder_name])
  • Add a remote on it (git add remote master origin)

How to contribute to a repository? (1st part)

  • First fork the repository, it will copy it on your GitHub account
  • Clone it on your local git
  • Go in the directory (cd whatever_you_called_it)
  • Add a remote on it (reference to the main repo in order to stay up-to-date) by doing : git remote add upstream url_of_the_main_repo
  • Fetch it ! (git fetch upstream)
  • Don't forget to read the README.md :)

As well before working on a team project, don't forget to checkout if something has changed, pull it.

How to contribute to a repository? (2nd part)

  • When you think you did a great job, do the commit process
  • After, go to your GitHub account
  • Then send a pull request (create a new branch before in order to do it)
  • Wait for admin validation :)
